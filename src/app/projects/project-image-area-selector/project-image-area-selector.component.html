Image Area Selector

A Javascript plugin to select an area of an image. I couldn't find a suitable plugin to use for a project, so I decided to create my own. It's not been optimised for mobile yet but it's in the road map!

If you'd like to use this it's available to download on npm or you can see the source code on Github.

Features

  • Specify height and width (min/max)
  • Use native or relative units
  • Maintain image aspect ratio or use given custom ratio
  • Download via npm

Stack

  • Javascript
  • HTML
  • CSS
