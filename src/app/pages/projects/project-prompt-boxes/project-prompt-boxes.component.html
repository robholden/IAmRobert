Prompt Boxes

A Javascript plugin to select an area of an image. I couldn't find a suitable plugin to use for a project, so I decided to create my own. It's not been optimised for mobile yet but it's in the road map!

If you'd like to use this it's available to download on npm or you can see the source code on Github.

Try it out

Permanent Toast Success Error Info Alert Confirm Prompt Clear

Features

  • Stack toast with a given limit
  • Permanent or timed toast
  • Option to manually close toasts
  • Toast as a success, warning or information style
  • Confirmation prompt
  • Dialogue prompt
  • Alert box
  • Download via npm

Stack

  • Javascript
  • HTML
  • CSS
