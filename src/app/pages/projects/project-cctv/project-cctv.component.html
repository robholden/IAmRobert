CCTV Gimmick

A Javascript plugin that shows a cctv camera that moves when a user scrolls up & down a web page. It will also blink when the user moves their cursor to pretend it is recording...

Right clicking on it will "lock" you out, and you will have to enter a pin (12345) to regain access. Or, just reload the page!

If you'd like to use this it's available to download on Github.

Features

  • Tracks scroll movement
  • "Records" motion
  • Locks out user when right clicking the camera

Stack

  • Javascript
  • HTML
  • CSS
