Snow Capture

Snow Capture is personal project I've been working for a while now. It is what I use to learn and test new tech & ideas I have. I started the project four years ago in PHP but I've since re-built the entire code base into C# and Angular 4.

Features

  • User authentication
  • Image uploading
  • Image geocode detection
  • Location lookup algarithm (using Google maps)

Stack

  • C# / .NetCore
  • Entity Framework
  • TypeScript
  • Angular 6
  • HTML5
  • CSS3/Sass
